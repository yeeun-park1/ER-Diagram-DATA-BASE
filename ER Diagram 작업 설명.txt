프로젝트 명 : ER Diagram & Data Base (MySQL 기반)

투입 인원 : 5명

프로젝트 소개 : 테이블 명세서와 ER Diagram을 제작하고 그에 따른 Data Base를 작성

공동 작업: 
	각각 생각해온 아이디어로 진행된 회의를 통해 프로젝트 기획

팀장 :
	모든 팀원이 개념을 이해하고 모두가 제작해볼 수 있도록 다른 팀과의 차별화
	분량 늘리기 : 카테고리 ( 호텔 / 항공 / 렌터카)
	팀원의 역량에 맞춰 2 명씩 소그룹으로 나눈 뒤 작업량 분배
	소그룹의 작업이 끝날 때마다 작업 결과 확인 후 개념 체크 및 설명

기능 구현
1. ER Diagram
	제작 방법 및 기본키와 외래키 설명
	관계 설명 : 1:1 , 1:N , M:N
	데이터 타입 설명 : VARCHAR INT DATE …
	ER Diagram은 관계를 살피기 좋은 표

2. 테이블 명세서
	제작한 ER Diagram을 활용하여 제작
	제작 방법 및 NULL 설명
	컬럼명과 널 허용 여부 등 미리 지정하기 좋은 표

3. MySQL
	제작한 2개의 표를 활용하여 데이터베이스 구현
	다양한 명령문 및 문법 설명

개발 도구 :

∙ 개발산출문서 – ER Diagram, 테이블 명세서 (Excel)
∙ DB - MySQL
